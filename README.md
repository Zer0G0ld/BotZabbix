# BotZabbix

# Instalação
Git : 
```git
git clone https://github.com/Zer0G0ld/BotZabbix.git
```

# Ambiente Virtual - AV
Criando ...
```
python3 -m venv venv

```

Ativando no Linux ou MacOS
```
source venv/bin/activate

```

Ativando no Windows
```
.\venv\Scripts\Activate
```

Desativando ...
```
deactivate

```

### Instalando dependências

```
pip install -r requirements.txt
```
